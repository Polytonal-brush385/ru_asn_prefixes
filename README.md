## 📦 What this is

ru_asn_prefixes gives you daily updated lists of IPv4 and IPv6 prefixes for Russian Autonomous Systems. It pulls data from RIPE NCC and groups it into ready-to-use files.

Use it when you need:
- IP lists for routing rules
- Split tunneling setup
- Firewall rules
- Network filtering
- Router import files

The files are meant for people who want current RU ASN prefix lists without building them by hand.
